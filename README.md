# Timetable Management System

A full-stack web application for managing and accessing academic timetables within an educational institution. The system provides dedicated interfaces for administrators, teachers, and students, enabling efficient timetable management through a centralized PostgreSQL database powered by Supabase.

## Tech Stack
- HTML
- CSS
- JavaScript
- Supabase
- PostgreSQL

## Key Features
- Role-based interfaces for Admin, Teacher, and Student
- Centralized timetable management
- Real-time database integration
- User-friendly web interface
